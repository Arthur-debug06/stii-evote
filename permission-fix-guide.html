üìã Step 1: Access InfinityFree File Manager

üìÅ Step 2: Navigate to Your Files

üîß Step 3: Change File Permissions

Method 1: Right-Click Menu (Most Common)

  1. Right-click on index.php
  2. Look for "Properties" or "Permissions" or "chmod"
  3. Click on it to open permissions dialog
  4. Change permissions to 644
  5. Click "Apply" or "Save"

Method 2: Properties Panel

  1. Click once on index.php to select it
  2. Look for a "Properties" panel (usually on the right)
  3. Find "Permissions" or "Mode" section
  4. Change the value to 644
  5. Press Enter or click "Apply"

Method 3: Toolbar Button

  1. Select index.php by clicking on it
  2. Look for a "Permissions" or "chmod" button in the toolbar
  3. Click it to open permissions dialog
  4. Set permissions to 644
  5. Click "OK" or "Apply"

üéØ Understanding Permissions

Permission Meaning Use For
644 Read/Write for owner, Read for others PHP files (.php), HTML files (.html)
755 Read/Write/Execute for owner, Read/Execute for others Directories (folders)
666 Read/Write for everyone Config files (some hosts)

üìù What the Numbers Mean

644 breaks down as:

  • 6 = Owner can read (4) + write (2) = 6
  • 4 = Group can read (4) only
  • 4 = Everyone can read (4) only
