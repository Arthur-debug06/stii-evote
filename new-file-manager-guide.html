üìã Step 1: Access Your Files

  1. Login to InfinityFree Dashboard
  2. Click "Manage" for your website
  3. Click "File Manager"
  4. Navigate to htdocs folder
  5. You should see your Laravel files: index.php, app/, vendor/, etc.

üîß Step 2: Change Permissions (New Method)

Method 1: Permissions Button (Most Likely)

  1. Click ONCE on index.php to select it
  2. Look at the toolbar buttons at the top
  3. Find and click "Permissions" button
  4. A dialog will open - change to 644
  5. Click "Apply" or "Save"

Method 2: Properties/More Button

  1. Select index.php
  2. Look for "Properties" or "More" or "..." button
  3. Click it to see additional options
  4. Find "Permissions" or "chmod"
  5. Set to 644

Method 3: Edit Button Alternative

  1. Select index.php
  2. Sometimes "Edit" button shows file properties first
  3. Look for permissions section in the edit dialog
  4. Change permissions before closing

üîç Can't Find Permissions Button?

If there's no obvious "Permissions" button, try these:

Look for: [Properties] [Details] [Info] [Advanced] [More] [Settings] [...]

The permissions option might be inside one of these menus!

üÜò Emergency Workaround

If you absolutely cannot find permissions settings:

  1. Create a new index.php using the file manager's "New File" or "Upload"
  2. Copy the content from your existing index.php
  3. Delete the old index.php
  4. Rename the new one to index.php

Sometimes newly created files have correct permissions automatically!
